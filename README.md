## Contribution

<details>
  <summary>Local development</summary>

  ```bash
  # Copy git repo
  git clone https://github.com/HardoWare/nuxt-ui-pack.git
  
  # Install dependencies
  npm install
  
  # Generate type stubs
  npm run dev:prepare
  ```
